# COMP1028---roticanAI
Appendix A: ReadMe  

Ensure you have installed:  

Visual Studio Code  

GCC (MinGW-w64 for Windows) 

C/C++ Extension by Microsoft 

Compile the C program: 

Open VS Code 

Open the folder that contains the .c files and all input files 

Open new terminal in VS Code 

Compile using gcc main.c -o program.exe 

Required Input Files: 

testToxic.txt – Unfiltered dataset (text file) 

testToxic.csv – Unfiltered dataset (CSV format) 

corurptedFile.txt – Used to test corrupted file detection 

emptyFile.txt – Used to test behavior with empty inputs 

stopWords.txt – List of stop words to remove 

toxicWords.txt – List of toxic words for filtering 

filteredWords.txt – Output of cleaned/filtered words 

analysisReport.csv – Final analysis report (output file) 

Note on dependencies: 

The program only uses standard C libraries 

No external libraries are required.
